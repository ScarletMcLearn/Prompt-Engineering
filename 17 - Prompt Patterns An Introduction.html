Prompt Patterns: An Introduction

Prompt patterns allow us to activate powerful mechanisms inside a Large Language Model. These models are trained on vast amounts of text and are highly sensitive to recurring linguistic and structural patterns. They use these patterns to decide how to behave and what type of response to generate.

That is why one of the key concepts in this course is the prompt pattern — a structured way of organizing words and instructions within a request.

What Is a Prompt Pattern?

A prompt pattern is a structured formulation that helps achieve a specific goal when interacting with a language model.

In other words, if we have a task we want the model to perform, a prompt pattern is a stable phrasing structure that increases the likelihood of obtaining the desired behavior.

It is not just about wording — it is about structure.

A Simple Example of a Pattern

Suppose we want to reliably trigger a specific continuation:

“If you give a mouse a cookie…”

This phrase is widely known from a popular children's book title. Because it frequently appears in training data, the model will very likely continue it with:

“…he’s going to ask for a glass of milk.”

If we repeatedly provide the same opening, we will typically receive the same or a very similar continuation. This gives us a degree of control and predictability.

The behavior is not perfectly deterministic — variations are always possible due to probabilistic generation — but we observe noticeable consistency.

This illustrates an important principle:

A prompt pattern is a structured linguistic trigger that activates a predictable continuation behavior.

Prompt Patterns for Practical Tasks

In real-world work, patterns are applied to practical objectives such as:

  • Forcing the model to answer only “Yes” or “No”

  • Generating concise summaries

  • Asking the model to interview you by asking questions

  • Ensuring specific elements are always included in responses

  • Enforcing a strict output format

Each of these tasks can be solved by adjusting the structure of the prompt — that is, by applying the appropriate pattern.

Why Does This Work?

Large Language Models generate text by predicting the next token based on learned patterns.

They have internalized thousands of linguistic structures during training. When we phrase a request in a way that matches familiar patterns, we activate those learned structures.

If our prompt aligns with a commonly observed pattern, the probability of obtaining the desired behavior increases.

That is why documenting and consciously using prompt structure is so important. We are not just asking questions — we are activating behavioral templates within the model.

Combining Patterns

As you move through this course, pay attention to situations where multiple patterns are used together.

Notice:

  • Which patterns are combined

  • In what order

  • Why that order matters

When designing your own prompts, do not ask:

“Which single pattern should I use?”

Instead ask:

“Which combination of patterns will produce the desired result?”

That is when you truly begin to control the model.
