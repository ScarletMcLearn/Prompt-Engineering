Question Refinement Pattern

A simple and highly effective way to improve your interaction with large language models like ChatGPT is to use the Question Refinement Pattern.

The idea is straightforward, but it often unlocks additional insights: what other prompt patterns could we embed to get more precise, high-quality answers?

The core concept is simple:
we instruct the model to suggest a better version of our question every time we ask one.

Why This Works

As users, we may not fully understand all the terminology, criteria, or structural elements needed for a strong question. The model, however, has seen vast amounts of text and has learned how well-formed questions in a given domain are typically structured.

Intuitively speaking:
we provide a general or vague question, and the model applies its understanding of effective linguistic patterns to refine it into something clearer and more precise.

Basic Pattern Structure

You can write something like:

“Whenever I ask a question, suggest a better version of the question and ask whether I want to use it instead of my original.”

You can also limit the scope:

  • Only for the next question

  • For the next three questions

  • From this point forward

A helpful improvement is to instruct the model to ask whether it should apply the refined version. This avoids manual copying and keeps the interaction natural.

Logic:

  • If you answer “yes” → the model uses the improved question

  • If “no” → it answers the original

Adapted Example (Culturally Neutral)

Original Example:

“Стоит ли мне поступать в МГУ?”

Why it needs adaptation:

MGU (Moscow State University) is culturally specific and not universally relevant for an international Udemy audience.

Replacement:

“Should I apply to a top-tier university?”

This keeps:

  • the decision-making context

  • the ambiguity of criteria

  • the need for refinement

But removes:

  • country-specific reference

  • institution-specific familiarity assumptions

Example Interaction

Initial vague question:

“Should I apply to a top-tier university?”

The model might refine it as:

“What factors should I consider when deciding whether to apply to a top-tier university, and how do they align with my academic goals, financial situation, and long-term career plans?”

Then it asks:

“Would you like to use this improved version instead?”

Why This Pattern Is More Than Just Rewriting

The refined version often reveals missing context:

  • What field of study?

  • What financial constraints?

  • What geographic preferences?

  • What long-term goals?

The pattern doesn’t just improve wording.
It exposes hidden assumptions and incomplete framing.

Formal Pattern Structure (for documentation)

To implement the Question Refinement Pattern, your prompt should include:

  1. From now on, every time I ask a question, suggest a better version of it.

  2. (Optional) Ask whether I want to use the improved version instead of the original.

Adapted Practical Examples

Diet-related example (already globally applicable)

Whenever I ask a question about diet, suggest a more precise version of the question that emphasizes healthy habits, sustainable lifestyle changes, and balanced nutrition. Then ask whether I want to use the improved version.

Business example (no cultural adjustment needed)

Whenever I ask about a business idea or launching a project, suggest a more specific version that clarifies target audience, market, budget, and objective. Then ask whether I want to use the refined version.
